## 项目简介

本项目是一个基于 Qt 6 的重症监护室（NICU）设备模拟系统，用于模拟多种医疗监护设备的工作流程和数据采集。系统包含四个主要医疗设备模块，并配备数据传输客户端进行串口数据模拟。

## 功能特性

### 📊 心电图仪
- 12导联心电图实时显示
- 历史数据加载与回放
- 网格背景与波形绘制
- 串口实时数据接收

### 🩺 血压仪
- 高压、低压、脉搏实时监测
- QWT曲线图表显示
- 历史趋势记录

### 👨‍⚕️ 血液透析仪
- 设备自检流程模拟
- 管道连接与清洗
- 预充与治疗流程
- 圆形进度条动画
- 紧急停止功能

### 💨 呼吸机
- **三种通气模式**：容量控制(VCV)、压力控制(PCV)、压力支持(PSV)
- **参数分类**：设定参数、监测参数、派生参数
- **参数锁定机制**：防止误操作
- **实时数据更新**：监测参数和派生参数动态刷新
- **报警灯指示**：绿/黄/红/橙四种状态

## 技术栈

- **框架**: Qt 6.11.1 (C++17)
- **UI框架**: Qt Widgets
- **串口通信**: Qt Serial Port
- **图表绘制**: QWT 6.3.0
- **构建工具**: qmake + MinGW 64-bit

## 快速开始

### 环境要求

- Qt 6.11.1 (MinGW 64-bit)
- QWT 库
- 虚拟串口驱动（如 Virtual Serial Port Driver）

### 编译步骤

1. 打开 Qt Creator，导入 `NICU.pro` 或 `Transmission.pro`
2. 配置构建套件为 Desktop Qt 6.11.1 MinGW 64-bit
3. 构建项目

### 运行步骤

1. 安装虚拟串口驱动，创建串口对（如 COM3 ↔ COM4）
2. 先运行 **Transmission** 客户端，连接相应串口
3. 再运行 **NICU** 主程序
4. 点击设备按钮进入对应模块

## 串口配置

| 设备 | 串口 | 波特率 |
|------|------|--------|
| 心电图 | COM1 | 115200 |
| 血透仪 | COM2 | 115200 |
| 血压仪 | COM3 | 115200 |
| 呼吸机 | COM4 | 115200 |
